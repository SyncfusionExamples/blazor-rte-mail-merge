# blazor-rte-mail-merge

A minimal Blazor sample demonstrating mail-merge capabilities using a Rich Text Editor component. This repository contains a small example application that shows how templates and data can be combined to produce personalized documents from within a Blazor UI.

## Overview

This sample illustrates the common mail-merge workflow in a Blazor application: define a template in a rich-text editor, provide a simple data source, and generate merged output for preview or export. The project is intended as a starting point for integrating mail-merge features into Blazor pages and for learning how template content and data can be programmatically combined.

## Features

- Template editing with a Rich Text Editor
- Simple data-driven merge into document templates
- Example UI for previewing merged results

## Prerequisites

- Visual Studio 2022 (or later)
- .NET SDK (6.0 or later)

## Setup & Running Steps

Installation

```bash
git clone https://github.com/SyncfusionExamples/blazor-rte-mail-merge
cd blazor-rte-mail-merge
```

Restore NuGet packages

```bash
dotnet restore
```

Run the application

```bash
dotnet run
```

## Troubleshooting & support

If you encounter issues, confirm that packages are restored, the project builds, and the license key is registered correctly. For more details about Syncfusion components consult the Syncfusion Blazor documentation.
